# 节奏调控快速开始

## 🎯 3 分钟快速上手

### 在 Web UI 中使用

1. 打开 http://localhost:3000/worlds
2. 点击"生成新世界"按钮
3. 在"叙事节奏 ⚡"下拉菜单中选择预设
4. 开始生成！

### 节奏预设速查表

| 预设 | 适用场景 | 句长 | 动作密度 | 描写占比 | 特点 |
|-----|---------|------|---------|---------|------|
| 🏃 **action** | 战斗、追逐 | 12字 | 90% | 20% | 短句密集、极度紧张 |
| 🏔️ **epic** | 仙侠、史诗 | 22字 | 60% | 50% | 变化丰富、场景宏大 |
| 📚 **literary** | 文艺、心理 | 28字 | 20% | 70% | 长句流畅、氛围浓厚 |
| 😱 **horror** | 恐怖、惊悚 | 16字 | 40% | 60% | 舒缓压抑、步步惊心 |
| 🔍 **detective** | 侦探、推理 | 20字 | 50% | 50% | 平稳清晰、逻辑性强 |
| ☀️ **slice_of_life** | 日常、校园 | 18字 | 30% | 50% | 温馨舒缓、对话丰富 |
| ⚖️ **balanced** | 通用场景 | 18字 | 50% | 40% | 各项均衡（默认）|

## 🎨 快速决策指南

### 你想要什么样的节奏？

**快节奏、紧张刺激** → 选择 `action`
- 适合：动作冒险、战斗、生死一线
- 效果：心跳加速、一口气读完

**慢节奏、细腻优雅** → 选择 `literary`
- 适合：心理描写、氛围营造、情感戏
- 效果：沉浸体验、回味无穷

**富有变化、史诗感** → 选择 `epic`
- 适合：仙侠、奇幻、宏大叙事
- 效果：跌宕起伏、气势磅礴

**不确定？** → 选择 `balanced`
- 万能选择，适合所有场景

## 💡 使用技巧

### 技巧 1: 根据小说类型选择

```
玄幻仙侠 → epic
科幻冒险 → action
恐怖惊悚 → horror
侦探推理 → detective
都市言情 → slice_of_life
```

### 技巧 2: 新手建议

- 第一次使用？选 **balanced**
- 觉得太慢？改用 **action** 或 **epic**
- 觉得太快？改用 **literary** 或 **slice_of_life**

### 技巧 3: 进阶用法

如果你想自定义节奏，可以通过 API 调用：

```bash
curl -X POST http://localhost:8000/api/worlds/generate \
  -H "Content-Type: application/json" \
  -d '{
    "novel_id": "novel-001",
    "seed": "dark survival",
    "pacing_preset": "action"  # 使用动作快节奏
  }'
```

## 🔧 常见问题

**Q: 节奏预设会影响什么？**
A: 影响句子长度、描写密度、动作频率、对话比例等，从句法到场景到事件，全方位控制节奏。

**Q: 可以中途改变节奏吗？**
A: 节奏配置在创建世界时设定，后续生成会遵循该配置。如需改变，需创建新世界。

**Q: 哪个预设最受欢迎？**
A: 对于仙侠类：**epic**；对于现代动作：**action**；不确定时：**balanced**

**Q: 节奏调控是否支持自定义？**
A: 是的！详见 `docs/features/PACING_CONTROL_GUIDE.md` 了解如何自定义每个参数。
