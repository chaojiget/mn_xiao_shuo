# 文风选择系统指南

## 🎯 快速概览

文风系统让你可以选择不同的写作风格，从网络小说到传统文学，应有尽有。

## 📚 可用文风预设

### 网络小说风格

#### 🔥 网文爽文 (web_novel_cool)
**适合**: 玄幻、仙侠、系统流

**特点**:
- 装逼打脸、爽点密集
- 四字成语多（"天赋异禀"、"横扫千军"）
- 第三人称限知视角
- 使用排比句式增强气势

**示例**:
```
他缓缓抬头,眼中寒光一闪。
"区区炼气期,也敢在本座面前放肆?"
剑气纵横,一剑破万法!
```

#### ☀️ 网文温情 (web_novel_warm)
**适合**: 都市、日常、恋爱

**特点**:
- 温馨日常、对话丰富
- 第一人称视角更亲切
- 用词简单,接地气
- 口语化表达

**示例**:
```
我推开咖啡厅的门,一股咖啡香扑面而来。
"来了啊。"她朝我笑了笑,眼睛弯成月牙。
我心里一暖,这就是平凡生活中的小确幸吧。
```

#### 🌑 网文黑暗 (web_novel_dark)
**适合**: 末世、黑暗流、无限流

**特点**:
- 阴暗压抑、写实残酷
- 镜头感强、画面感突出
- 描写丰富,营造氛围
- 用词高级

**示例**:
```
腐臭的血肉覆盖着破碎的街道。
他踩在碎玻璃上,发出令人牙酸的嘎吱声。
远处传来低沉的嘶吼——又是一只变异体。
```

### 传统文风

#### 📜 古典雅致 (classical_elegant)
**适合**: 历史、武侠、传统仙侠

**特点**:
- 文言白话混合
- 使用典故,典雅庄重
- 古典句式（倒装、省略）
- 华丽描写

**示例**:
```
月华如水,洒满庭前。
少年白衣胜雪,立于梅下,手持长剑。
剑光一闪,恍若惊鸿,倏忽已至眉间。
"好剑法!"众人齐声赞道。
```

#### 🏛️ 古风白话 (archaic_vernacular)
**适合**: 穿越、宫斗、古代背景

**特点**:
- 古风韵味但易读
- 四字词组多
- 适中用词
- 第三人称限知

**示例**:
```
宫墙深深,红烛摇曳。
她轻掀珠帘,步履款款而入。
"娘娘驾到!"太监尖声唱道。
殿内诸人纷纷跪拜,不敢抬头。
```

#### 📖 现代文学 (modern_literary，默认)
**适合**: 现代背景、纯文学

**特点**:
- 现代白话,文学性强
- 适合大众阅读
- 用词适中
- 平衡描写

**示例**:
```
雨后的城市街道泛着湿润的光泽。
他撑着伞走在人行道上,思绪飘向远方。
橱窗里的灯光温暖而明亮,像是在召唤着什么。
```

### 特殊风格

#### 🌸 诗意优美 (poetic_beauty)
**适合**: 散文、唯美向、情感细腻

**特点**:
- 诗化语言,意境悠远
- 使用典故和排比
- 第一人称
- 丰富描写

**示例**:
```
我站在桥上看风景,看风景的人在楼上看我。
明月装饰了你的窗子,你装饰了别人的梦。
那年的夏天,阳光洒落在发梢,我们的青春就这样流淌。
```

#### 🎬 镜头感惊悚 (cinematic_thriller)
**适合**: 悬疑、惊悚、侦探

**特点**:
- 画面感强,镜头语言
- 极简描写,聚焦关键
- 用词适中
- 第三人称限知

**示例**:
```
门缓缓打开。
吱呀——
黑暗中,一双眼睛注视着他。
他的手,在颤抖。
```

#### 😄 口语化幽默 (vernacular_humorous)
**适合**: 轻松搞笑、日常向

**特点**:
- 口语化表达
- 幽默诙谐,接地气
- 第一人称
- 极简描写

**示例**:
```
我跟你讲啊,这事儿真不赖我。
你信不信?反正我是信了。
结果呢?结果就被老板逮个正着。
完蛋,这下彻底凉凉了。
```

## 💡 如何选择

### 按小说类型选择

```
玄幻仙侠 → 网文爽文 或 古典雅致
都市言情 → 网文温情 或 现代文学
末世恐怖 → 网文黑暗 或 镜头惊悚
穿越宫斗 → 古风白话
历史武侠 → 古典雅致
轻松搞笑 → 口语幽默
唯美散文 → 诗意优美
悬疑推理 → 镜头惊悚
```

### 快速决策树

**你想要什么风格?**

- **网络小说风** → 选择三种网文风格之一
  - 爽文向 → 网文爽文
  - 温情向 → 网文温情
  - 黑暗向 → 网文黑暗

- **传统文学风** → 选择传统文风
  - 古代背景 → 古典雅致 或 古风白话
  - 现代背景 → 现代文学

- **特殊需求** → 选择特殊风格
  - 要意境 → 诗意优美
  - 要画面感 → 镜头惊悚
  - 要搞笑 → 口语幽默

## 🔧 使用方法

### Web UI 使用

1. 打开 http://localhost:3000/worlds
2. 点击"生成新世界"
3. 在"写作文风 ✍️"下拉菜单选择
4. 开始生成！

### API 调用

```bash
# 列出所有文风
curl http://localhost:8000/api/worlds/writing-style/presets

# 获取特定文风配置
curl http://localhost:8000/api/worlds/writing-style/presets/web_novel_cool

# 使用文风创建世界
curl -X POST http://localhost:8000/api/worlds/generate \
  -H "Content-Type: application/json" \
  -d '{
    "novel_id": "novel-001",
    "seed": "仙侠修炼",
    "writing_style_preset": "web_novel_cool"
  }'
```

## 🎯 文风参数详解

每个文风包含以下配置:

### 1. 文风类型 (style_type)
- `classical`: 古典文言
- `archaic`: 古风白话
- `modern`: 现代白话
- `poetic`: 诗意优美
- `vernacular`: 口语化
- `literary`: 文学性
- `cinematic`: 镜头感

### 2. 词汇难度 (vocabulary_level)
- `simple`: 简单易懂
- `moderate`: 适中
- `advanced`: 高级复杂
- `archaic`: 古雅

### 3. 修辞手法
- `use_metaphor`: 比喻
- `use_personification`: 拟人
- `use_parallelism`: 排比
- `use_allusion`: 典故

### 4. 句式特点
- `classical_syntax`: 古典句式（倒装、省略）
- `four_character_phrases`: 四字成语/词组
- `poetic_language`: 诗化语言

### 5. 叙事视角 (narrative_pov)
- `first`: 第一人称（我）
- `third_limited`: 第三人称限知
- `third_omniscient`: 第三人称全知

### 6. 描写风格 (description_style)
- `minimalist`: 极简
- `balanced`: 平衡
- `lush`: 丰富
- `ornate`: 华丽

## 💬 常见问题

**Q: 文风和节奏有什么区别?**
A:
- **文风**: 控制用词、句式、修辞等,决定"怎么说"
- **节奏**: 控制句长、动作密度等,决定"说多快"

**Q: 可以混合使用吗?**
A: 每个世界只能选择一种文风预设,但可以自定义参数组合。

**Q: 哪个文风最适合网络小说?**
A:
- 仙侠 → 网文爽文
- 都市 → 网文温情
- 末世 → 网文黑暗

**Q: 古典雅致会不会太难读?**
A: 古典雅致用词较古雅,如果追求易读性,选择"古风白话"更合适。

## 🚀 高级技巧

### 技巧 1: 文风与节奏搭配

```
网文爽文 + 快节奏 = 经典网文套路
古典雅致 + 慢节奏 = 传统武侠风
镜头惊悚 + 快节奏 = 紧张刺激
```

### 技巧 2: 根据场景切换

虽然一个世界只有一种主文风,但可以通过prompt微调:
- 战斗场景 → 强调动作、镜头感
- 感情戏 → 强调诗意、细腻
- 日常场景 → 强调口语、对话

### 技巧 3: 新手推荐

- 第一次使用? → **现代文学** (默认,最稳妥)
- 写网文? → **网文爽文** (最受欢迎)
- 写传统? → **古风白话** (易读但有韵味)
